# Switch
#javascript
```jsx
const dia = 0; //0: Domingo...

switch (dia) { //Hace un ===
    case 0:
        console.log('Domingo');
        break;
    case 1:
        console.log('Lunes');
        break;
    case 2: 
        console.log('Martes');
        break;
    default:
        console.log('No es domingo, ni lunes ni martes');
        break;
}
```
